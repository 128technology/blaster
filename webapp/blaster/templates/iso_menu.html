Managing ISOs

This page allows you to setup official 128T ISOs and prepare them for PXE booting onto whitebox hardware

  • List Available ISOs - Clicking this link will query the official 128T yum repo for available ISOs using a previously staged yum certificate (See the "Manage Certificates" menu for more information). Clicking on any ISO in this list will begin the process to download the selected ISO and stage it for PXE booting. Progress of the download can be checked in the "List ISOs" menu.
  • List ISOs - View the list of ISOs that the blaster has attempted to stage for PXE boot, see which ISO is currently active for PXE booting, and see the status of ISO downloads in progress
  • Upload ISO - Directly upload an official 128T ISO instead of downloading directly from the yum server
