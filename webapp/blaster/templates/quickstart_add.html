Managing Quickstarts

This page can be used to upload quickstarts that were manually downloaded from a conductor rather than having the blaster pull the quickstart from a conductor's API.

  • Conductor - A free form label to identify which conductor the quickstart is associated with. This value will appear in the quickstart list. This value does not need to match any conductor name in the Manage Conductor list.
  • Router Name - The name of the router. This is also a free form label that will appear in the quickstart list. Please ensure this correctly matches the router the quickstart was pulled from. This value will not be validated against the contents of the quickstart.
  • Description - A free form description that will appear in the quickstart list
