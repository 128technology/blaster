Managing Bootstrap Scripts

Scripts can be used to execute commands on the node before or after the bootstrapping procedure is completed. This menu makes no distinction between a pre-bootstrap-script or a post-bootstrap-script, that distinction is made when associating the scripts with the ISO through the Manage ISOs menu. The blaster does no validation of these scripts to ensure they will run successfully, these scripts should be validated prior to uploading.

  • Add Script - Upload a new script to the blaster
  • List Scripts - List all scripts available to the blaster
