Managing Conductors

This table will list all conductors that have been successfully added to the system. Clicking the "List Nodes" link for a conductor will retreive information about all 128T nodes in that conductor's configuration. This will be displayed on a new page with each node's name, router name, and asset ID shown in a table. A link will be available next to each node allowing you to download the node's quickstart into the blaster.

The "Delete" link next to a conductor will remove this entry from the blaster, including the authentication token.
