Managing Scripts

This table lists all of the scripts available to use for pre-and-post bootstrapping. Entries can be removed from the blaster by clicking the "Delete" link.

Deleting a script does not remove it from the kickstart of any ISOs it is currently associated with. Please also remove the script association through the ISO menu.

Clicking the "View" link will show the contents of the script. Modifying the script from this view is not supported, if a script needs to be modified please delete the current script and upload the modified version. Then re-associate the script with any ISOs it was previously associated with.
