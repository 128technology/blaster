Managing Quickstarts

Quickstart files are used to put an initial configuration on a node so that it can, at a minimum, reach the conductor. Quickstarts can be retrieved by the blaster through a conductor's API following the instructions available in the Manage Conductors menu.

  • Add Quickstart - This option can be used to manually upload a quickstart file to the blaster rather than having the blaster retreive it from a conductor API
  • List Quickstarts - View and manage quickstarts that have been retreived from conductor APIs or uploaded manually
