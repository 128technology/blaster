Managing Conductors

A new conductor can be added to the blaster on this page. Using the provided credentials, the blaster will authenticate to the conductor and create a token. This token will be saved in the blaster's database and used in future communications with this conductor.

  • Conductor Name - A free form name you can use to describe this particular conductor
  • Conductor URL - The URL used to contact the conductor using the format https://<conductor IP>[:<conductor port>]
  • Conductor Username - The username to use when generating an authentication token
  • Conductor Password - The user's password
