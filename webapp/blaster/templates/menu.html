About the Blaster

The blaster is designed to function as a PXE boot server to use for installing official 128T OTP ISOs on whitebox hardware. In addition, the blaster can function as a bootstrap server offering 128T quickstart files via REST per the official 128T bootstrapping process. The official home of the blaster is on the 128T GitHub site. Below you will find details about the menu options found above.

  • Manage Certificates - Manage your 128T Yum certificate needed to access published ISOs
  • Manage ISOs - Download ISOs from the 128T yum server, view ISOs already downloaded, and change the active ISO currently used for blasting systems
  • Manage Conductors - Setup Access to Conductors in order to download quickstarts via API
  • Manage Quickstarts - View quickstarts currently ready for use with the blaster and upload new quickstarts manually
  • Manage Nodes - View systems that have been blasted, make associations between a specific device and a quickstart file following the OTP process
  • Manage Passwords - Setup passwords to override ISO system default passwords
